# Code of Conduct

This project follows version 2.1 of the **Contributor Covenant**, the most
widely adopted open-source community code of conduct.

Read it here: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

If you experience or witness behavior that violates the Covenant, please
contact **LXB Studio LLC** at <LXBStudioLLC@gmail.com>. Reports will be reviewed
and responded to within 7 days, and treated as confidential.

## Enforcement

Maintainers may remove comments, commits, or contributions that do not align
with the Covenant. In repeat or severe cases, maintainers may temporarily or
permanently ban a contributor from interaction with the project.

## Scope

This Code applies in all project spaces (issues, pull requests, discussions,
release channels) and when an individual is representing the project in public
spaces (e.g., a conference talk or a social media post tagging the project).
