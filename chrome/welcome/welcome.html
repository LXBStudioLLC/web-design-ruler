Web Design Ruler

Essential design tools for web designers — pixel-perfect measurements, font identification, and professional color picking.

Your three tools

🎨

Color Picker

Pick colors from any element, image, canvas, or video. Copy as HEX, RGB, or HSL, and build palettes as you go.

Aa

Font Detector

Hover over text to identify the rendered font family, weight, size, line height, and color. Copy the full CSS in one click.

📐

Measurement Tool

Click and drag to measure any element. Get width, height, diagonal, and area — hold Shift to snap to a 10px grid.

Keyboard shortcuts

  • Alt+Shift+R — Open the Web Design Ruler popup
  • Alt+Shift+P — Activate the Color Picker
  • Alt+Shift+F — Activate the Font Detector
  • Alt+Shift+M — Activate the Measurement Tool

You can also right-click any page and choose one of the “… with Web Design Ruler” menu items.

Privacy by design

Web Design Ruler runs entirely offline. No telemetry, no analytics, no remote assets, no network requests. Everything you pick, measure, or save stays in your browser’s local storage on this device.

Tools operate only on the active tab you invoke them on, and the extension never reads page content unless you explicitly trigger a tool.
